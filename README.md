# Applied AI Engineering

A hands-on repository documenting my journey to becoming an Applied AI Engineer by building AI systems from first principles instead of relying on high-level abstractions.

The goal of this repository is to understand **how modern AI applications work internally** by implementing every layer step by step, then gradually adopting production-ready frameworks like LangChain.

## What I've Built

### Phase 1 — Manual RAG

* Document Loading
* Recursive Text Chunking
* Embedding Generation
* Vector Database (FAISS)
* Semantic Search
* Prompt Construction
* Local LLM Integration (Ollama)

### Phase 2 — LangChain Fundamentals

* Document Loaders
* Text Splitters
* Embedding Models
* Vector Stores
* Retrievers
* Prompt Templates

### Phase 3 — LCEL (LangChain Expression Language)

* Runnable Interface
* RunnableLambda
* RunnablePassthrough
* RunnableParallel
* Prompt Pipelines
* Output Parsers
* End-to-End LCEL RAG Pipeline

## Project Philosophy

Instead of treating AI frameworks as black boxes, this repository focuses on understanding **why each abstraction exists** by first implementing the underlying concepts manually and then refactoring them using LangChain.

Every major concept is built, tested, and understood before moving to the next abstraction.
